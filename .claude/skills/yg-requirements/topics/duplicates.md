# Topic — Game duplicates (3.6)

Source: <https://yandex.ru/dev/games/doc/ru/requirements/3/6>

## Rule
> Игра не является полной или частичной копией другой игры из каталога и не дублирует другую игру данного разработчика.

Sequels / new versions must include substantive changes — new mechanics, content, art.

## Cannot check from code
This is a moderation check against the catalog. Mark ❓ Unverifiable in audits and list manual checks:

1. Search the Yandex Games catalog for the game name and confirm no near-identical clone exists.
2. If publishing a sequel, list the substantive changes vs the previous version.
3. Do not re-publish the same build under multiple slugs.
